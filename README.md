# 墨池

一款基于 Tauri 2.0 + React 的 Windows 本地优先小说写作工具，专为中文长篇小说创作设计。

## 技术栈

| 层面 | 技术 |
|------|------|
| 桌面框架 | Tauri 2.0 (Rust) |
| 前端 | React 19 + TypeScript 5.8 |
| 状态管理 | Zustand 5 |
| 编辑器 | TipTap (ProseMirror) |
| 样式 | Tailwind CSS 4 |
| 测试 | Vitest + happy-dom |

## 核心功能

### 写作编辑
- **富文本编辑器**：基于 TipTap，支持标题、加粗、斜体、列表、引用、分隔线、排版优化扩展
- **中文排版**：首行缩进两字符（可开关）、自定义字号/行高/段间距/页边距/编辑区最大宽度
- **自动整理格式**：一键标点全角化、引号配对、空白与连续空段收敛
- **逐键草稿缓冲**：每次按键镜像至 localStorage 草稿缓冲区，崩溃/断电后可恢复
- **自动保存**：每 3 秒将编辑器内容落盘（通过原子写入：先写 .tmp 再 rename，保证文件永不截断）
- **关闭窗口强制落盘**：Rust 侧接管窗口关闭事件，先通知 JS 刷新所有待保存内容，再销毁窗口并退出进程
- **历史版本**：写作中每 5 分钟自动快照（有改动时），每章最多保留 20 个，可按时间线预览并一键回滚
- **自定义写作目标**：整书总目标 + 每章独立目标（默认跟随全局设置），底部状态栏显示进度条

### 文件 I/O 安全
- **自定义存储位置**：作品保存位置可在设置中指定任意文件夹，支持一键迁移既有内容
- **路径白名单授权**：Rust 侧动态维护授权目录列表，webview 发起的任何文件 I/O 均须通过 authorize_path 校验，支持 MSIX 虚拟化路径重定向
- **每作品独立文件夹**：按"作品名-id/project.json"+"chapters/卷序号-章节名.md"组织，重名章节自动加数字后缀消歧
- **原子写入**：所有文件写入走 .tmp 文件、fsync、同卷 rename，保证写入过程永不产生截断文件
- **多重安全守卫**：删除作品目录需验证包含 project.json；导出文件走 OS 原生对话框，Rust 侧发放一次性写入授权；路径 ID 均校验防注入

### 项目与章节
- **卷/章两级结构**：支持拖拽排序卷、拖拽归类章节到不同卷
- **章节状态**：草稿 / 修改中 / 待校对 / 已完成，章节树中显示状态圆点
- **右键菜单**：章节项支持右键重命名、删除、切换状态等操作
- **导出**：单章导出 Markdown / TXT；整本导出 HTML / Markdown / TXT（走 OS 原生保存对话框）

### 全书搜索与替换
- **全文搜索** (Ctrl+Shift+F)：跨全部章节检索标题与正文，按卷分组展示，点击跳转到对应章节
- **全文替换**：支持逐个替换和全部替换，基于 HTML 文本节点的安全映射（匹配跨标签时跳过而非破坏 HTML 结构），替换前检查是否过期防止滞后写入

### 写作笔记
- **左侧栏笔记区**：每作品独立笔记，支持文件夹归类、置顶、拖拽移动至文件夹
- **笔记文件夹管理**：新建/重命名/删除文件夹，删除文件夹时笔记移回未归档

### 设定词典
- **词条化写作设定**：人物卡、地点、势力、物品、概念五类预设，支持自定义分类
- **词条详情视图**：左侧词条列表可搜索筛选，右侧详情面板编辑内容
- **别名系统**：每个词条支持多个别名

### 写作统计
- **字数统计**：总字数、今日新增字数、当前章字数、整书进度条
- **写作时间**：有效写作时长统计（仅统计实际输入时间）
- **底部状态栏**：实时显示字数、进度、写作时长、保存状态

### 界面
- **作品库**：书籍封面卡片视图 / 紧凑列表视图可选，鼠标悬停显示操作按钮
- **左侧栏**：三页签切换（章节目录 / 写作笔记 / 设定词典），宽度可拖拽调整
- **右侧栏**：大纲视图 / 历史版本
- **专注模式**：一键隐藏双侧栏，顶部工具栏仅悬停时浮出，右下角保留退出按钮
- **深色模式**：浅色/深色/跟随系统，深色为暖调墨色低蓝光配色
- **5 种主题色**：墨棕 / 黛蓝 / 松绿 / 胭脂 / 紫檀 / 青蓝
- **5 种纸张质感**（浅色）：米白 / 纯白 / 羊皮纸 / 护眼绿 / 青瓷色
- **快捷键自定义**：应用级快捷键（保存、新建章节、搜索、侧栏、专注模式等）均可在设置中修改
- **编辑器字体**：正文和界面各自独立选择字体（均为系统已装字体，零下载）

## 快捷键

| 快捷键 | 动作 | 说明 |
|--------|------|------|
| Ctrl+S | 保存 | 立即将当前章落盘 |
| Ctrl+N | 新建章节 | 在当前卷末尾新建 |
| Ctrl+Shift+F | 全书搜索 | 跨全部章节检索标题和正文 |
| Ctrl+B | 目录侧栏 | 打开/关闭左侧栏（目录页签） |
| Ctrl+Alt+O | 大纲面板 | 打开/关闭右侧栏大纲 |
| Ctrl+Shift+D | 专注模式 | 进入/退出专注模式 |
| F11 | 窗口全屏 | 切换 Tauri 窗口全屏状态 |
| Esc | 退出专注模式 | 专注模式下按 Esc 退出 |

## 数据存储

```
<应用数据目录>/                    <作品保存目录>/
├── registry.json                  ├── <作品名>-<id>/
│                                  │   ├── project.json
│                                  │   ├── chapters/
│                                  │   │   ├── 1-第1章.md
│                                  │   │   └── 1-第2章.md
│                                  │   ├── notes/
│                                  │   │   └── <作品id>.json
│                                  │   └── dictionary/
│                                  │       └── <作品id>.json
│                                  └── inkwell-content-roots.txt
```

- 应用数据（settings + registry.json）：系统 AppData 目录
- 作品内容：默认位于应用数据目录，可在设置中自定义至任意文件夹
- 章节文件命名：{卷序号}-{章节名}.md，重名自动加数字后缀
- 草稿缓冲：localStorage（每次按键写入，成功落盘后清除）
- 历史快照：chapters/{id}.snapshots/{时间戳}.html

## 开发

```bash
npm install          # 安装依赖
npm run tauri dev    # 启动桌面应用（开发模式，含热更新）
npm run dev          # 纯浏览器预览（localStorage 兜底存储）
npm test             # 运行单元测试（Vitest + happy-dom）
npm run tauri build  # 构建正式 Windows 安装包（NSIS + MSI）
```

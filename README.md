# 墨池

一款基于 Tauri 2.0 + React 的 Windows 本地优先小说写作工具。

## 功能特性

- 📝 **富文本编辑器**：基于 TipTap，支持标题、加粗、斜体、列表、引用、分隔线
- 📚 **章节管理**：卷 / 章两级结构，章节与卷均可拖拽排序、状态标识
- 🎨 **中文排版优化**：首行缩进两字符（可开关）、自定义字号 / 行高 / 段间距 / 页边距 / 编辑区宽度
- 💾 **数据安全**：每 3 秒自动保存 + 逐键草稿缓冲（崩溃/断电可恢复）+ 关闭窗口强制落盘
- 🕘 **历史版本**：写作中每 5 分钟自动快照（有改动时），每章最多保留 20 个，可预览 / 回滚
- 🔍 **全书搜索**：Ctrl+Shift+F 跨全部章节检索标题与正文，按卷分组，点击跳转
- 📊 **写作统计**：总字数 / 目标进度、今日新增、有效写作时长、本章进度条
- 📤 **导出**：单章 Markdown / TXT；整本 HTML / Markdown / TXT
- 🗒️ **写作笔记**：左侧栏笔记区，记录人物设定、灵感、伏笔，随项目自动保存
- 📖 **设定词典**：人物卡、地点、势力等词条化管理，支持分类筛选与全文搜索
- 🧘 **专注模式**：一键隐藏双侧栏，退出后恢复原布局；顶部热区一碰即唤出工具栏
- 🎭 **个性化主题**：5 种主题色（墨棕/黛蓝/松绿/胭脂/紫檀）+ 3 种纸张质感（米白/羊皮纸/护眼绿）
- 🗂 **存储位置自定义**：作品内容位置可改，支持一键迁移既有内容
- 🌗 **主题**：浅色 / 深色 / 跟随系统；深色为暖调墨色低蓝光配色，夜间写作更护眼

## 快捷键

编辑器内：加粗 Ctrl+B、斜体 Ctrl+I、标题 Ctrl+Alt+1/2/3、撤销 Ctrl+Z、重做 Ctrl+Y。

应用：保存 Ctrl+S、新建章节 Ctrl+N、全书搜索 Ctrl+Shift+F、目录侧栏 Ctrl+B、
大纲面板 Ctrl+Alt+O、专注模式 Ctrl+Shift+D、窗口全屏 F11。

## 开发

```bash
npm install
npm run dev        # 浏览器预览（localStorage 兜底存储）
npm run tauri dev  # 桌面应用
npm test           # 单元测试
npm run build      # 类型检查 + 前端构建
```

## 数据存储

- 应用数据（设置 + 作品索引 registry.json）：系统数据目录
- 作品内容（projects/<id>.json、chapters/<id>.md）：数据目录，或在设置中自定义
- 写作笔记：notes/<作品id>.json；设定词典：dictionary/<作品id>.json（均随作品一同迁移）
- 历史快照：chapters/<id>.snapshots/<时间戳>.html
- 草稿缓冲与写作统计：localStorage（随写随存，落盘后自动清理）
